List of fields:

white_list:ip whitelist.(ex:192.168.1.134,192.168.1.156)

port: Port. (ex: 12345)

resetOption: Reset time. (>= 120(ms) ex :120)

priority_telco: Telco code. (ex: VIN,MOB,VTT. Telco's list: VIN,MOB,VTT) 

priority_loc: Location code. (ex:MB,MN.Location's list:MB,MN,MT)

note:Your note.

key_password: Password you want to set (If you do not want to set a password, you can skip this field).

ex:

{
  "white_list": "192.168.1.134,192.168.1.156",
  "port": 12340,
  "resetOption": 120,
  "priority_telco": "VTT,VIN",
  "priority_loc": "MB,MT",
  "note": "test",
  "key_password": "123456"
}
